# Звартноц — сайт ансамбля

Статический лендинг армянского народного ансамбля танцев «Звартноц»: о коллективе, концертах, галерее, контактах и ссылках на Instagram. Светлая и тёмная тема оформления.

## Структура проекта

| Путь | Назначение |
|------|------------|
| `index.html` | Главная страница |
| `css/main.css` | Стили |
| `js/main.js` | Тема, меню, карусель, плавная прокрутка |
| `assets/images/logo.png` | Логотип |
| `.nojekyll` | Отключает Jekyll на GitHub Pages (чистый HTML/CSS/JS) |

Сборщики и Node.js не используются.

## Локальный просмотр

Из корня репозитория:

```bash
python3 -m http.server 8080
```

Откройте в браузере: [http://127.0.0.1:8080/](http://127.0.0.1:8080/)

## Публикация на GitHub Pages

Для личного сайта на `github.io` репозиторий на GitHub обычно называют **`ваш-логин.github.io`**, ветка — **`main`**, файлы сайта — в **корне** репозитория.

1. Создайте такой репозиторий на GitHub (без README, если уже есть локальный коммит).
2. Скопируйте URL репозитория: зелёная кнопка **Code** → HTTPS (например `https://github.com/…/….git`).
3. В папке проекта выполните `git remote add origin <вставьте URL>` и `git push -u origin main`.
4. На GitHub: **Settings → Pages** → источник **Deploy from a branch**, ветка **main**, папка **`/ (root)`**.
5. Сайт появится по адресу `https://ваш-логин.github.io/` (подставьте свой логин).
